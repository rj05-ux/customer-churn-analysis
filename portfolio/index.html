Customer Churn Analysis and Prediction

Built an end-to-end machine learning pipeline to analyze customer behavior and predict churn using a dataset of 6000 records. The project focuses on identifying key drivers of churn and improving customer retention strategies.

Problem Statement

Predict customer churn and identify behavioral patterns that lead to customer loss, enabling businesses to take proactive retention measures.

Tech Stack

  • Python
  • Pandas, NumPy
  • Matplotlib, Seaborn
  • Scikit-learn (Random Forest)

Pipeline

Data Loading → Cleaning → Feature Engineering → EDA → Model Building → Evaluation → Insights

Model Performance

  • Accuracy: 98%
  • Precision (Churn): 87%
  • Recall (Churn): 68%

Key Insights

  • Engagement is the strongest predictor of churn
  • Low satisfaction increases churn risk
  • Behavior matters more than pricing

Model Improvement

Resolved data leakage and handled class imbalance using weighted Random Forest, improving recall from 61% to 68%.
